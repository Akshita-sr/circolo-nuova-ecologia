Sito web ufficiale del **Circolo Nuova Ecologia di Genova** — associazione ambientalista fondata il 6 maggio 1986.

> *Per un ambientalismo sociale a Genova dal 1986.*

Il sito è pubblicato su **GitHub Pages**:

## 🚀 Attivare GitHub Pages

Per pubblicare il sito su GitHub Pages:

1. Vai alla pagina del repository su GitHub
2. Clicca su **Settings** (Impostazioni)
3. Nel menu laterale, clicca su **Pages**
4. Sotto "Source", seleziona **Deploy from a branch**
5. Scegli il branch **`main`** (o `master`) e la cartella **`/ (root)`**
6. Clicca **Save**

Dopo qualche minuto, il sito sarà disponibile all'indirizzo:
`https://<username>.github.io/circolo-nuova-ecologia/`

## 🛠️ Tecnologie utilizzate

- **HTML5** semantico con attributi ARIA per l'accessibilità
- **CSS3** puro (variabili CSS, Flexbox, Grid, animazioni)
- **JavaScript vanilla** (nessun framework esterno)
- **Google Fonts**: Montserrat + Open Sans
- Nessuna dipendenza esterna: compatibile al 100% con GitHub Pages
